# 点滴ドロップ計算器 PRO (IV Drip Calculator)

看護師さんの現場での負担を軽減し、安心・確実な滴下管理をサポートするために、プログラミング学習の一環として作成されたアプリケーションです。

## 🌟 プロジェクトの想い
テレビの特集で、看護師さんの深刻な人手不足や、看護学校の減少を知りました。
「日々命と向き合う看護師さんの課題を、最新の技術で少しでもお力になれないか？」
そんな想いから、ベテランデザイナーの視点を取り入れた、現場特化型の計算器を開発しました。

## ✨ 主な機能
- **基本計算**: 輸液量と予定時間から、1分間の滴下数を即座に算出。
- **残量からの逆算**: 現在の残量と滴下ペースから、終了予定時刻を予測。
- **滴下リズムガイド**: 画面の点滅に合わせてクレンメを調整できるメトロノーム機能。
- **終了10分前アラート**: ブラウザ通知機能による、うっかりミスの防止。
- **ナイトモード**: 暗い病棟でも患者さんを刺激しない、目に優しい配色。
- **オフライン対応 (PWA)**: 電波の届かない場所でも動作。

## 📱 使い方
1. 設定（点滴セット：20滴/60滴）を確認。
2. 輸液量と予定時間を入力。
3. 表示された「滴下数」と「リズムガイド」を参考に、実機で調整を行う。
4. 終了時刻を確認し、必要に応じて通知アラートをオンにする。
